# Across QA Test Suites

Across QA Test Suites is an automated test suite created to ensure the quality, stability, and reliability of the Across application and its related components.

---

## Purpose

The purpose of this test suite is to standardize and automate QA processes by providing a reliable and maintainable test framework.

This repository aims to:
- Reduce manual testing effort
- Detect defects early in the development lifecycle
- Validate critical user flows
- Support regression and smoke testing
- Enable safe and confident releases through automated testing

---

## Test Scope

The test suites included in this repository cover:
- Smoke tests
- Functional tests
- Regression tests
- End-to-end (E2E) test scenarios

Tests are designed to be reusable, scalable, and suitable for different environments.

---

## Technologies Used

- Java  
- Selenium WebDriver  
- Maven  
- JUnit / TestNG  
- Assertions
- Reports
- JDBC
- Rest Assured
- IntelliJ IDEA


The project is designed to be CI/CD friendly and can be integrated with tools such as GitHub Actions or Jenkins.
